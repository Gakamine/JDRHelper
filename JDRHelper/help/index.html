  • Personnages

  • Comment changer les informations des personnages ?

    Dans le menu de la fenêtre de JDRHelper, sélectionnez "Personnages". Dans le menu déroulant, cliquez sur "Editer section". Vous pouvez modifier les informations et laisser des sections vides.

  • Comment enregistrer les informations ?

    Lorsque vous créer un personnage, à l'instant où vous cliquez sur "confirmer", les informations reliées au personnage modifié ou créé seront enregistrées.

  • Comment supprimer un personnage ?

    Cette action n'est pas disponible sur le logiciel. En revanche, vous pouvez modifier les informations d'un personnage pour le remplacer.

  • Comment ajouter un inventaire ?

    Pour ajouter un inventaire à un personnage vous devez d'abord avoir enregistré une fois les informations. Si vous n'avez pas encore créer un objet dans la section "Objet" du menu de la fenêtre, l'option "Gérer l'inventaire" n'apparaitra pas lors de la modification de la fiche personnage.

  • Scénarios

  • Comment supprimer un scénario ?

    Cette action n'est pas disponible. En revanche, vous pouvez modifier un scénario pour le remplacer. Vous ne pouvez pas remplacer un scénario PDF importé.

  • Est-il possible d'importer un scénario ?

    Vous pourrez uniquement importez un scénario au format PDF. Vous retrouverez ce scénario en accès rapide dans le menu de la fenêtre "Scénario".

  • Objets

  • Comment supprimer un objet ?

    Cette action n'est pas disponible. En revanche, vous pouvez modifier un objet pour le remplacer.

  • Cartes

  • Puis je supprimer une carte ?

    Cette action n'est pas disponible.

  • Comment masquer la carte ?

    Vous pouvez redimensionner la taille de la fenêtre pour ainsi ne laisser apparaitre que le menu de la fenêtre en accès rapide.

  • Ennemis

  • Comment supprimer un ennemi ?

    Cette action n'est pas disponible. En revanche, vous pouvez modifier les informations d'un ennemi pour le remplacer.

  • Sauvegardes

  • Comment créer une sauvegarde ?

    Dans le menu "fichier", sélectionnez "Exporter sauvegarde".

  • Comment importer une sauvegarde ?

    Dans le menu "fichier", sélectionnez "Importez sauvegarde". Les sauvegarde sont enregistrées dans le dossier "saves" dans le dossier racine du logiciel.

  • Comment modifier une sauvegarde ?

    Cette fonction n'est pas disponible.
